UNCURL

UNCURL is a unified framework for various unsupervised learning tasks with single cell transcriptomic data that accounts for the read distribution due to the sequencing process. It can be used to perform a variety of tasks such as:

  • Clustering
  • True transcriptomic state estimation
  • Dimensionality reduction
  • Lineage estimation
  • Semi-supervision using any kind of binarizable prior information
